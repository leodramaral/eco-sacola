
  # Design EcoSacola Manaus

  This is a code bundle for Design EcoSacola Manaus. The original project is available at https://www.figma.com/design/ddedSb0XLgwkgtRcBY1wgI/Design-EcoSacola-Manaus.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Cross-platform encoding and line endings

  This repository is configured to use UTF-8 and LF line endings by default (Ubuntu/Linux first), while keeping Windows `.bat` and `.cmd` scripts with CRLF when needed.

  If you already cloned the project before these settings, run:

  ```bash
  git add --renormalize .
  ```

  Then commit the normalization changes once.
